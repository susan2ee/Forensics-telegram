# MessageExtractor_aVersion

A Telegram Web A memory message extractor based on parsing TL schema definitions and scanning filtered memory dumps.

---

## Description

This tool is designed to extract and parse Telegram messages from memory dump files. \
It leverages TL (Type Language) schema definitions from the official Telegram Web A repository to decode messages.

---

## Usage

```bash
python -m message_extractor_aVersion.py <memory_dump>
